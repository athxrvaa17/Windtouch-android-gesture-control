# WindTouch - Android Gesture Control 🖐️📱

**WindTouch** is an innovative Android application that enables touchless interaction using real-time hand gesture recognition. Built with **Kotlin** and powered by **Google's MediaPipe**, this app detects hand landmarks through your phone's camera, allowing users to control or interact with the interface entirely through the air.

### 🚀 Key Features
* **Real-Time Hand Tracking:** Utilizes MediaPipe to detect 21 3D hand landmarks with high precision and low latency.
* **Touchless Interaction:** Enables gesture-based controls directly on your Android device.
* **CameraX Integration:** Smooth and efficient camera feed processing for optimal performance.
* **Modern Android Architecture:** Built using Kotlin and standard Android development practices.

### 🛠️ Tech Stack
* **Language:** Kotlin
* **IDE:** Android Studio
* **Machine Learning / Vision:** Google MediaPipe (Hand Landmarker)
* **Camera API:** CameraX
